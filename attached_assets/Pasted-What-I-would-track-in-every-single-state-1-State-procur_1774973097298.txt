What I would track in every single state
1) State procurement

Start with each state’s official procurement portal from NASPO’s directory. That is the cleanest way to catch state RFPs, RFIs, bid tabs, awards, contract vehicles, and vendor contacts. For Occu-Med, I’d watch for keywords like occupational health, employee health, drug testing, pre-employment physicals, corrections health, DOT/CDL medical, public safety exams, vaccination, audiology, dental, and mobile medical.

2) State legislatures and bill tracking

This is one of the strongest state-level intel streams. NCSL maintains 50-state bill-tracking databases, a current legislative session calendar, and topic-specific databases for health workforce, public-sector workforce, health costs/coverage/delivery, and occupational licensing. NCSL also notes that every state legislature offers free public bill-status access online, and legislatures or legislative chambers in at least 43 states offer customizable tracking/notification tools. That means you can monitor state laws that could change screening rules, scope of practice, telehealth, licensing portability, public employee medical standards, or workforce shortages.

3) Governor’s office and major state agencies

USAGov’s state directory is the best “seed map” for every state because it links to the official state government site, governor, health department, emergency management agency, corrections department, and other major agencies. For the portal, I’d use this as the state landing-page source-of-truth, then crawl outward into each agency newsroom, press release page, or bulletin page.

4) State health departments

ASTHO’s member directory gives you the leadership map for state health departments across the country, and USAGov links directly to the official state health departments. This is where you catch public-health alerts, immunization policy changes, outbreak notices, workforce guidance, rural-access issues, and health-system strain that could affect clinics, scheduling, or client operations.

5) State labor / workforce agencies and WARN signals

This is a big one for Andy. DOL maintains state WARN contacts, and NASWA represents all 50 state workforce agencies. That gives you a way to track layoffs, dislocated-worker activity, labor-market stress, training pushes, and workforce-agency updates by state. For Occu-Med, that is useful because WARN and workforce signals can point to new prospecting targets, regional demand shifts, or employer distress before a contract opportunity becomes obvious.

6) State licensing and discipline

For provider/network intelligence, I’d absolutely include state medical boards and nursing boards. FSMB maintains a state medical-board contact directory and notes that physician licensure and disciplinary information is available through its Physician Data Center; NCSBN provides board-of-nursing contacts for U.S. members. This is useful for provider risk checks, network integrity, disciplinary-action monitoring, and validating professional oversight by state.

7) State emergency management and disaster disruption

USAGov provides a state emergency-management directory for all states. This matters more than it sounds: disaster declarations, evacuations, severe-weather disruptions, and state emergency actions can affect appointment completion, clinic operations, travel feasibility, and local medical capacity. I would make this a core operational signal, not an afterthought.

8) State OSHA-plan / workplace safety signals

OSHA lists which states run OSHA-approved state plans, including states that cover both private-sector and state/local-government workers and states that cover only state/local government workers. For Occu-Med, this is relevant because state-level workplace safety enforcement and standards can shape employer demand, public-sector compliance, and the kinds of occupational-health services being emphasized locally.

9) State insurance departments

NAIC’s state insurance-department directory is worth adding. It helps surface state insurance regulators, which matters when you want to watch coverage changes, telehealth/payment policy, network adequacy issues, complaint patterns, and state insurance pressure points that can affect provider behavior or reimbursement environments.

10) State corrections

USAGov also provides state corrections departments by state. I would include this because corrections systems often generate signals around inmate healthcare, contracted medical services, staffing shortages, facility expansions, and health-related procurements that can be relevant either directly or as adjacent market intelligence.
